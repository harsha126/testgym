# IRON ADDICTS Brand Styling

## Overview

To access IRON ADDICTS's official brand identity and style resources, use this skill.

**Keywords**: branding, corporate identity, visual identity, post-processing, styling, brand colors, typography, IRON ADDICTS, visual formatting, visual design

## Brand Guidelines

### Colors

**Main Colors:**

- Royal Blue: `#0544A4` - Primary background and major content areas
- Light Gray: `#E6E6EA` - Body text color
- White: `#FFFFFF` - Heading text color and primary button backgrounds

**Accent Colors:**

- Black: `#000000` - Primary button text and footer links
- Light Gray (Footer): `#F4F4F4` - Secondary backgrounds

### Typography

- **Headings**: Old Standard TT (with serif fallback)
- **Body Text**: Gudea (with Georgia, serif fallback)
- **UI Text**: GD Sherpa (with Helvetica, Arial fallback)

## Features

### Smart Font Application

- Applies Old Standard TT font to large headings (H1, H2)
- Applies Gudea font to descriptive body text
- Uses GD Sherpa for interactive UI elements (buttons, menus)

### Text Styling

- Headings: Old Standard TT, often italicized for logo/brand impact
- Body text: Gudea
- High contrast with White (`#FFFFFF`) headings and Light Gray (`#E6E6EA`) body text on Royal Blue (`#0544A4`) background

### Shape and Accent Colors

- Primary Buttons feature stark contrast (White background with Black text)
- Secondary interactions use inverse styling (Royal Blue background with White text)

## Technical Details

### Font Management

- Primary heading fonts are served via external font providers
- Provides automatic fallbacks to generic serif or sans-serif families based on use case

### Color Application

- Uses precise HEX color values for exact brand matching across web assets
